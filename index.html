The 1.21.4+ Shift Explained

In Minecraft 1.21.4, Mojang removed CustomModelData overrides from models/item/paper.json.

It now relies on a Data-Driven component system located in items/paper.json. This tool does the heavy JSON conversion automatically. Simply select the 1.21.4 target.

Spawn Command (1.21.4+):

/give @p paper[custom_model_data={floats:[1]}]
